# Build a WhiteList DApp | Smart Contract

## Description

This is the smart contract part of the project. Here we are implementing principles of Solidity to handle the saving and recignizing whitelisters on the blockchain.

## Tools Used

- Solidity
- Hardhat
- Alchemy
- Goerli Testnet
- Ethers.js

## Concepts applied

- Constructors
- Functions
